# 破局 - 历史切面中的抉择

## 项目简介

一个纯前端的历史文字冒险游戏，让玩家代入历史人物面临绝境时刻，考验"破局之力"。

## 项目结构

```
poju-game/
├── src/              # 源代码
│   ├── index.html    # 游戏主文件（当前版本）
│   ├── css/          # 样式文件
│   ├── js/           # 逻辑脚本
│   └── data/         # 关卡数据
├── assets/           # 静态资源
│   ├── images/       # 图片素材
│   ├── fonts/        # 字体文件
│   └── audio/        # 音效文件
├── docs/             # 文档
│   ├── changelog.md  # 更新日志
│   ├── roadmap.md    # 开发路线图
│   └── design.md     # 设计文档
├── scripts/          # 工具脚本
│   ├── deploy.sh     # 部署脚本
│   └── build.sh      # 构建脚本
├── versions/           # 历史版本存档
│   ├── v1.0/         # 初始版本
│   ├── v1.1/         # 卡片式重构
│   └── v2.0/         # 未来版本
└── README.md         # 本文件
```
