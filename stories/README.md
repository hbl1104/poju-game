# 破局 · 故事衍生工程

本目录存储「破局」游戏各关卡的故事文本，采用加缪式存在主义口吻撰写，便于进行微信公众号、小红书、抖音短视频等平台的衍生创作。

## 文件结构

```
stories/
├── README.md          # 本文件
├── jingke.md          # 药囊掷荆轲（战国 · 前227年）
├── caoshuang.md       # 富家翁之阱（曹魏 · 249年）
├── yuyi.md            # 御医之刀（西晋 · 280年代）
└── xilong.md          # 扬州之狐（西晋 · 301年）
```

## 每个故事文件的结构

每个 `.md` 文件包含以下部分：

1. **基础信息**：时代、地点、主角、困局、破局道具
2. **加缪式故事正文**：分六节撰写
   - 第一节：场景铺陈，引入荒诞感
   - 第二节：困局揭示，主角的处境
   - 第三节：三种荒诞（三种选择的后果）
   - 第四节：破局路径的详细展开
   - 第五节：历史与哲学的反思
   - 第六节：破局者说，提炼核心思想
3. **衍生创作素材**：
   - 短视频文案（30秒）
   - 小红书图文标题（3个）
   - 公众号文章标题（3个）

## 风格指南

### 加缪式口吻的核心特征

1. **荒诞感**：强调人的行动与世界的沉默之间的对峙
2. **存在主义**：关注个体在绝境中的选择，而非历史必然性
3. **冷峻的叙述**：不煽情，不评判，让事实自己说话
4. **哲学追问**：每个故事都指向一个更深层的问题——人在荒诞中如何自处
5. **反英雄**：主角不是英雄，是普通人，在普通中做出不普通的选择

### 写作原则

- 不美化历史，不丑化人物
- 不给出标准答案，只呈现选择的后果
- 每个失败路径都有价值，都是对人类处境的揭示
- 历史注脚与哲学反思并重

## 使用方式

### 微信公众号

直接使用「公众号文章标题」+「故事正文」的前四节，约2000-3000字。

### 小红书

使用「小红书图文标题」+「短视频文案」，配合游戏截图或历史插画。

### 抖音短视频

使用「短视频文案」，配合历史画面或游戏画面，时长30-60秒。

### 播客/音频

使用完整故事正文，约10-15分钟朗读时长。
